# Ledger — Social CPA Study Platform

The social study platform built for CPA candidates who take passing seriously.

## Features

- 📊 **Study Dashboard** — track hours, streaks, exam pacing, and weekly goals
- 🏠 **Social Feed** — share sessions, give props, comment, follow other candidates
- 🏆 **Leaderboard** — compete with your cohort on weekly hours
- 🏛️ **Study Clubs** — create or join public/private study groups with live chat
- 💬 **Direct Messages** — message other candidates directly
- ⚡ **Challenges** — Easy to Elite difficulty challenges that auto-track from sessions
- 🎓 **Exam Tips** — real AICPA pass-rate data, strategy guides, score windows
- 📅 **Study Calendar** — visual calendar with session logs and exam date markers
- 🔥 **Achievement System** — 12 unlockable badges tied to real study behavior
- ▶️ **Live Session Timer** — start, pause, recap, and post sessions to your feed

## Premium (Ledger Pro)

- 🤖 AI Study Coach — personalized daily focus and weak-spot alerts
- 📊 Advanced Analytics — topic heatmaps, peak-time analysis, pass probability
- 🎯 Full Exam Simulator — timed mock exams with adaptive MCQs + TBS
- 📅 Smart Study Schedule — auto-generated daily plan synced to exam dates
- 🛡️ Streak Shield — protect your streak if you miss a day
- 📝 Weekly PDF Progress Reports
- Unlimited Study Clubs
- Custom per-section goals + data export

## Tech

Single-file HTML/CSS/JS app. No build tools, no dependencies, no server required. All state stored in `localStorage`.
